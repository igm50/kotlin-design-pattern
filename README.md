# Design Patterns sample by Kotlin

## このリポジトリについて

Kotlinで記述したGoFデザインパターンのサンプル。

## 注意

Kotlinで記述しているため、一部のデザインパターンや実装メソッドを除いている場合がある。
例えば、SingletonはKotlinの基本機能として提供されているため除いている。
